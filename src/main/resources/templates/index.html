Добро пожаловать на проект Info21 v2.0 Web!

Данный проект - это web-приложение с использование MVC-фреймворка. Здесь реализован web-интерфейс для проекта SQL2. Приложение поддерживает осуществление CRUD-операций, импорт/экспорт таблиц, осуществление разработанных на прошлом шаге операций/функций через графический интерфейс, а также логирование действий пользователя.
Об авторах: проект выполнили tyuuki и dumfred. Ученики школы 21.
Раздел "Данные" содержит CRUD-операции по всем таблиц, а также импорт/экспорт в CSV формат.
Раздел "Операции" содержит все возможные для вызова запросы из проекта SQL2.
